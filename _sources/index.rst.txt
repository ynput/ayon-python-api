Getting Started
===============

.. code-block:: text
   :caption: Install latest version from PyPi

       pip install ayon-python-api

.. code-block:: text
   :caption: Install from Github sources (Alternatively)

       git clone git@github.com:ynput/ayon-python-api.git
       cd ayon-python-api
       pip install .

.. code-block:: text
   :caption: Ensure installed properly by printing ayon_api version

       python -c "import ayon_api ; print(ayon_api.__version__)"
